# Contributing to IPCop-Alpine

Thank you for your interest. This project is currently in a pre-alpha state and is not actively seeking large-scale external contributions.

### Bug Reports
Please use the GitHub Issue tracker to report bugs. Include:
* Step-by-step instructions to reproduce the issue.
* Any relevant logs or error messages.

### Pull Requests
Small fixes or improvements are welcome. 
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the project's existing style (POSIX-compliant shell, strict Perl).
3. Provide a clear description of the change in your PR.
